# S3 Assignment Project

This repository contains my AWS S3 assignment files.

## Description
This project demonstrates hosting a static website on AWS S3. It includes:

- `index.html` — The main page of the website.
- `error.html` — Custom error page for S3 bucket.

## S3 Website URL
You can view the website here: [S3 Website Link](http://essa-cloudlab.s3-website-us-east-1.amazonaws.com)

## How to use
1. Open the `index.html` file in your browser or visit the S3 website link above.
2. The `error.html` page will be displayed when an invalid URL is accessed.
